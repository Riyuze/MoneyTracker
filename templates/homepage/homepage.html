ManeTorakka is a personal project I started to learn about a Python-based web framework called Django. Source code can be found on my github.

Features

Although ManeTorakka is still a work in progress, it is built with several features that enchances the user's experience.

Charts

ManeTorakka includes charts of your financial summary. Visually compare your data, understand yourself better.

Output Files

ManeTorakka provides several formats to print out your financial summary. The formats provided are Excel, CSV, and PDF.
